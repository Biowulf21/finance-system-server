# INSTALLATION
## Requirements
> Composer - Composer 1.10.x

> Php - Php 7.4.x

> Mysql - MySQL 5.7.x

## Installation Guide
### Clone the project
> `git@github.com:Biowulf21/finance-system-server.git`
### Change directory
> `cd finance-system-server`
### Install dependencies
> `composer install`
### Run server
> `php artisan serve`
